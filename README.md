# Kamico.store — Frontend

## Структура
- `/` — магазин для клиентов
- `/admin` — панель администратора

## Деплой на Vercel

1. Загрузите эту папку на GitHub
2. Зайдите на vercel.com → New Project
3. Выберите репозиторий
4. Framework: **Create React App**
5. Нажмите Deploy

## Локальный запуск
```
npm install
npm start
```
